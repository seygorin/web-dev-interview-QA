## Semantic HTML

Семантический HTML — это подход к разметке веб-страниц, при котором используются теги, отражающие смысл содержимого. Такие теги помогают браузерам, поисковым системам и вспомогательным технологиям (например, скринридерам) лучше понимать структуру и назначение различных частей страницы. Это повышает доступность сайта и улучшает его SEO.

### Examples of semantic tags

- `<header>` — верхняя часть страницы или раздела, обычно содержит логотип, навигацию, заголовки.
- `<nav>` — область для основной навигации по сайту.
- `<main>` — основное содержимое страницы, уникальное для каждой страницы.
- `<article>` — самостоятельный, независимый блок контента (например, статья, пост в блоге, новость).
- `<section>` — логически обособленный раздел внутри страницы или статьи.
- `<aside>` — дополнительная информация, боковая панель, сноски.
- `<footer>` — нижний колонтитул страницы или раздела, часто содержит контакты, копирайт, ссылки.

**Пример полной структуры страницы с использованием семантических тегов:**

```html
<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="UTF-8" />
    <title>Пример семантической разметки</title>
  </head>
  <body>
    <header>
      <h1>Мой сайт</h1>
      <nav>
        <ul>
          <li><a href="#home">Главная</a></li>
          <li><a href="#about">О нас</a></li>
          <li><a href="#contact">Контакты</a></li>
        </ul>
      </nav>
    </header>
    <main>
      <section>
        <h2>Добро пожаловать!</h2>
        <p>Это пример использования семантических тегов в HTML5.</p>
      </section>
      <article>
        <h2>Последние новости</h2>
        <p>Сегодня мы изучаем основы HTML5 и семантической разметки.</p>
      </article>
      <aside>
        <h3>Полезные ссылки</h3>
        <ul>
          <li>
            <a href="https://developer.mozilla.org/ru/docs/Web/HTML"
              >Документация HTML</a
            >
          </li>
          <li>
            <a href="https://html5book.ru/html-html5/">HTML5 для начинающих</a>
          </li>
        </ul>
      </aside>
    </main>
    <footer>
      <p>&copy; 2024 Мой сайт. Все права защищены.</p>
    </footer>
  </body>
</html>
```

В этом примере `<header>` содержит заголовок сайта и навигацию, `<main>` — основное содержимое, разделённое на `<section>` (приветствие), `<article>` (новости) и `<aside>` (боковая панель с полезными ссылками). Внизу страницы находится `<footer>` с копирайтом.

### Non semantic tags

Для сравнения, несемантические теги, такие как `<div>` и `<span>`, не несут смысловой нагрузки и используются только для группировки или стилизации элементов. Их стоит применять, когда нет подходящего семантического тега.

```html
<div class="container">
  <span class="highlight">Текст, выделенный цветом</span>
</div>
```

В этом примере `<div>` используется для группировки элементов, а `<span>` — для выделения части текста. Они не сообщают браузеру или поисковику о смысле содержимого.


## Basic examples of HTML elements

### Headings and paragraphs

Заголовки используются для обозначения структуры документа, а параграфы — для обычного текста. В HTML есть шесть уровней заголовков (`<h1>`–`<h6>`), где `<h1>` — самый важный.

```html
<h1>Заголовок первого уровня</h1>
<h2>Заголовок второго уровня</h2>
<p>Это обычный абзац текста.</p>
```

Заголовки помогают разбивать текст на логические части, а параграфы делают текст читаемым и структурированным.

### Lists

Списки позволяют структурировать информацию в виде маркированных (`<ul>`) или нумерованных (`<ol>`) пунктов.

```html
<ul>
  <li>Пункт маркированного списка</li>
  <li>Еще один пункт</li>
</ul>

<ol>
  <li>Первый пункт нумерованного списка</li>
  <li>Второй пункт</li>
</ol>
```

Маркированные списки используются для перечисления, а нумерованные — для последовательностей или инструкций.

### Links and images

Ссылки (`<a>`) позволяют переходить на другие страницы или сайты, а изображения (`<img>`) — отображать графику.

```html
<a href="https://example.com">Перейти на сайт</a>
<img src="logo.png" alt="Логотип компании" />
```

Атрибут `href` указывает адрес ссылки, а `src` — путь к изображению. Атрибут `alt` обязателен для изображений и нужен для доступности.

## Responsive images

Отзывчивые изображения автоматически подстраиваются под размер экрана и плотность пикселей устройства, что важно для мобильных пользователей и современных дисплеев.

### Using the `<picture>` tag

Тег `<picture>` позволяет задавать разные изображения для разных условий (например, ширины экрана). Это удобно для оптимизации загрузки и отображения изображений на разных устройствах.

```html
<picture>
  <source media="(min-width: 1000px)" srcset="large.jpg" />
  <source media="(min-width: 600px)" srcset="medium.jpg" />
  <img src="small.jpg" alt="Описание изображения" />
</picture>
```

В этом примере браузер выберет подходящее изображение в зависимости от ширины экрана пользователя.

### Using the `srcset` attribute for different screen densities

Атрибут `srcset` позволяет указать несколько версий одного изображения для разных плотностей пикселей (например, для Retina-экранов).

```html
<img
  src="image-1x.jpg"
  srcset="image-1x.jpg 1x, image-2x.jpg 2x"
  alt="Отзывчивое изображение"
/>
```

Браузер автоматически выберет наиболее подходящее изображение в зависимости от устройства пользователя.

### Styling images for responsiveness

Чтобы изображения не выходили за пределы контейнера и корректно масштабировались, используют такие стили:

```css
.responsive-image {
  max-width: 100%;
  height: auto;
  display: block;
}
```

Эти стили позволяют изображению уменьшаться вместе с размером окна браузера, сохраняя пропорции.

## Additional examples for practice

### Tables

Таблицы используются для представления табличных данных. В HTML таблица состоит из строк (`<tr>`), ячеек заголовков (`<th>`) и обычных ячеек (`<td>`).

```html
<table>
  <caption>
    Расписание занятий
  </caption>
  <thead>
    <tr>
      <th>День</th>
      <th>Время</th>
      <th>Предмет</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Понедельник</td>
      <td>10:00</td>
      <td>Математика</td>
    </tr>
    <tr>
      <td>Вторник</td>
      <td>12:00</td>
      <td>Физика</td>
    </tr>
  </tbody>
</table>
```

`<caption>` задаёт заголовок таблицы, `<thead>` — область заголовков, `<tbody>` — основное содержимое.

### Forms

Формы позволяют собирать данные от пользователя. В HTML формы состоят из различных элементов ввода: текстовых полей, email, кнопок и т.д.

```html
<form>
  <label for="name">Имя:</label>
  <input type="text" id="name" name="name" />

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" />

  <button type="submit">Отправить</button>
</form>
```

Атрибут `for` у `<label>` связывает подпись с соответствующим полем ввода по `id`. Кнопка отправляет данные формы на сервер или обрабатывается скриптом.
